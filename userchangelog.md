# Parch Linux Installer - User Changelog

## New Installer Experience (2026-05-18)

### What’s New
- The installer now has a fully redesigned interface with a cleaner, more modern look.
- Setup steps are easier to follow, with clearer titles and guidance on each screen.
- Navigation between steps is more consistent and better organized.

### Improved Setup Flow
- Welcome screen now gives clearer readiness information before installation.
- Language selection is easier to access and change.
- Region and timezone setup is simplified and more direct.
- Keyboard setup now has a clearer model, layout, and variant flow.
- User account setup has a cleaner form layout.
- Summary screen is easier to review before applying disk changes.

### Better Feedback During Input
- Username and hostname fields now provide clearer validation feedback.
- Password entry flow is improved with better inline feedback for mismatches.
- Form status is visually clearer while filling in required details.

### Stability and Compatibility Improvements
- Better keyboard/input reliability when running the installer in Wayland-based live sessions.
- Updated connectivity check target for a more distro-relevant online check.

### Visual Updates
- Refreshed slideshow and branding visuals during installation.
- Updated installer imagery for a more polished presentation.
